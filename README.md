# Home Decoor Website

## Project Description

Home Decor Hub is a React-based home decor website that allows users to browse featured products and add items to a waitlist for future updates or availability. It features a clean and responsive UI, dynamic product display, and interactive elements for improved user experience, demonstrating a front-end focused e-commerce platform for showcasing home decor items.
